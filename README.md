# Pasarela de Pago

Proyecto de Node.js/Express para simular una pasarela de pago.

## Tecnologías
Node.js + Express
HTML/CSS

## CI/CD
Este repositorio usa GitHub Actions (.github/workflows) para desplegar 
automáticamente en Render cada vez que se hace push a la rama main.

## Cómo correr localmente
\`\`\`
npm install
node server.js
\`\`\`
